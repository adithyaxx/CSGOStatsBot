## Getting Started
[Add telegram bot](https://t.me/csgo_stats_bot)

## Usage

Commands:

```
/get - retrieve player's stats
/getbanner - retrieve player's stats banner
```

Examples:
```
/get 76561198063387072
/get FlyingPenguin
/getbanner FlyingDinosaur
```
